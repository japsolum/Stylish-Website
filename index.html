Internet basics

What is the internet

The internet is basically a bunch of computers that communicate with eachother. They use browsers like chrome or Firefox to send command protocols to a server at another location in order to receive the web pages they are requesting. Those pages can include anything from pictures and movies to basic text, but all are comprised of html which is the language the computer reads so it knows what to display.

Understanding how a computer thinks

Computers are very literal. They have no ability to interpret what it thinks you mean. But basically have to be told like a child exactly what to do, and how to do it. Programming Languages

HTML

The structure of a webpage is built using html. Html is the language computers use to determine what is displayed on a web page. It is comprised of a series of opening and (usually) closing tags that form together to create elements. Elements can also be thought of as the overall idea of what you are telling the computer to do. These elements can be broken down further into categories of inline or block. Block elements single themselves out by basically creating an invisible box around the content that's inside.

CSS

CSS iss the style of a page. It can take the elements from html and change color, weight, size, and positioning of your elements. You can use it to modify inline elements, blocks, or entire classes all at the same time. Using css on your page is like using Microsoft word to type something instead of notepad.
